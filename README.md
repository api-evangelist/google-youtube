# YouTube Data API

The YouTube Data API v3 lets you incorporate YouTube functionality into your own application. You can use the API to fetch search results, manage videos, playlists, channels, subscriptions, and access activity data. It supports reading and writing YouTube data using RESTful operations with OAuth 2.0 authentication.

## Artifacts

- **OpenAPI**: [openapi/youtube.yml](openapi/youtube.yml) - OpenAPI 3.1.0 specification for the YouTube Data API v3.
- **JSON Schema**: [json-schema/youtube.json](json-schema/youtube.json) - JSON Schema (draft 2020-12) for YouTube video resources.
- **JSON-LD**: [json-ld/youtube.jsonld](json-ld/youtube.jsonld) - JSON-LD context mapping YouTube resources to Schema.org vocabulary.
