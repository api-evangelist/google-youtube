# YouTube Data (google-youtube)

The YouTube Data API v3 lets you incorporate YouTube functionality into your own application. You can use the API to fetch search results, manage videos, playlists, channels, subscriptions, and access activity data. It supports reading and writing YouTube data using RESTful operations with OAuth 2.0 authentication.

## APIs

### YouTube Data API v3

The YouTube Data API v3 provides access to YouTube data such as videos, playlists, channels, comments, subscriptions, and search results. It enables developers to build applications that interact with YouTube content and user data.

- **Human URL:** [https://developers.google.com/youtube/v3](https://developers.google.com/youtube/v3)
- **Base URL:** `https://youtube.googleapis.com/youtube/v3`
